# SPA 학습 데이터셋

현대자동차그룹 SPA(Speaking Proficiency Assessment) 영어시험 학습용 데이터.

## 시험 개요

- **시간**: 13분
- **형식**: 비대면 실시간 화상, 1:1 원어민 인터뷰
- **점수**: 96점 만점 / 8등급
- **목적**: 사내 승진, 해외 주재원 선발 평가

## 유형 구성

| 유형 | 이름 | 형식 | 시간 |
|------|------|------|------|
| 1 | Business Casual | 일상/개인 질문 응답 | 약 3분 |
| 2 | Opinion | 의견 표현 (논리/근거) | 약 3분 |
| 3 | Visual Description | 그래프/사진 묘사 및 비교 | 약 3분 |
| 4 | Passage Summary | 지문 듣고 60초 내 요약 | 약 3분 |

## 데이터 파일

| 파일 | 내용 | 개수 |
|------|------|------|
| `type1_business_casual.json` | 유형1 질문 + 모범답안 | 100 |
| `type2_opinion.json` | 유형2 질문 + 논점 + 모범답안 | 100 |
| `type3_visual.json` | 유형3 그래프/사진 + 묘사 답안 | 100 |
| `type4_summary.json` | 유형4 지문 + 요약 답안 | 100 |
| `mock_exams.json` | 모의고사 세트 (유형1~4 각 1문항) | 50 |

## 데이터 출처

- 공개된 SPA 후기, 학원 샘플 문제, 실제 수험자 복원 문제
- SPA 시험 형식과 난이도에 맞춰 패턴 기반으로 추가 생성
- 모든 데이터는 학습용이며 실제 기출과 무관

## SPA 등급 체계 (96점 만점, 8등급)

| 등급 | 점수 범위 |
|------|----------|
| Lv 1 | 0 ~ 15점 |
| Lv 2 | 16 ~ 24점 |
| Lv 3 | 25 ~ 34점 |
| Lv 4 | 35 ~ 49점 |
| Lv 5 | 50 ~ 64점 |
| Lv 6 | 65 ~ 74점 |
| Lv 7 | 75 ~ 84점 |
| Lv 8 | 85 ~ 96점 |

## 문제 난이도 기준

| 난이도 | 권장 목표 등급 | 응답 길이 | 어휘 수준 |
|--------|--------------|----------|----------|
| Easy | Lv 4~5 | 3-4문장 | 기본 |
| Medium | Lv 6~7 | 5-7문장 | 비즈니스 |
| Hard | Lv 7~8 | 8문장 이상 | 고급/전문 |
